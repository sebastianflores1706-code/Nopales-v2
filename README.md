# Nopales v2

Sistema de Administración de Espacios Públicos.

## Módulos

- Espacios
- Reservaciones
- Calendario
- Pagos
- Documentos
- Incidencias
- Mantenimiento
- Usuarios
- Reportes

## Roles del sistema

### Administrador
- Gestiona espacios
- Aprueba reservaciones
- Registra pagos
- Genera contratos
- Administra incidencias
- Visualiza reportes

### Ciudadano
- Solicita reservaciones
- Consulta estado de solicitudes
- Visualiza documentos

## Estados de reservación

- pendiente_revision
- aprobada
- rechazada
- cancelada
- en_uso
- finalizada
